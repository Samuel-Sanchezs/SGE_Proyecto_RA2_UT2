# 09 — Creación de base de datos de prueba
En mi caso el servidor ha sido creado en un dispositivo distinto, si estas ejecutando la pagina en el mismo puedes sustituir **tu dirrecion ip** por **localhost**
1. Accede a `http://"tu direccion ip":8069`.  
![Pagina web](/retos/Reto_03_Instalacion_Odoo_Linux_Samuel_Sanchez_Sandoval/assets/img/09-creacion_bd_prueba/odoo.png)
2. Crea una **base de datos nueva** con email/contraseña admin.  
    Recuerde que la contraseña y demas son las que tendra que usar posteriormente
3. Selecciona módulos iniciales si procede.  
    Una vez dentro recomiendo instalar la aplicacion de la pagina web para gestionarla, en caso de querer una.  
![Dashboard](/retos/Reto_03_Instalacion_Odoo_Linux_Samuel_Sanchez_Sandoval/assets/img/09-creacion_bd_prueba/dashboard.png)  

> Resultado esperado: BD de prueba creada y primer acceso.
