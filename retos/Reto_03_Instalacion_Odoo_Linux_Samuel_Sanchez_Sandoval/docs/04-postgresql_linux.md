# 04 — PostgreSQL en Linux

1. Instala PostgreSQL desde repos:
   ```bash
   sudo apt install postgresql
   ```
   Realizamos una instalación de la base de datos en el servidor 
2. Verifica el servicio:
   ```bash
   sudo systemctl status postgresql
   ```
   Mediante este comando podras ver el estado del servidor, en el ejemplo se puede ver que esta activo:  
   ![Status PostgreSQL](/retos/Reto_03_Instalacion_Odoo_Linux_Samuel_Sanchez_Sandoval/assets/img/04-postgresql_linux/status_postgre.png) 

> Resultado esperado: PostgreSQL instalado y activo.
