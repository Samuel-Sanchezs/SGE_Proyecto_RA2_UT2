# 03 — Descarga del instalador de Odoo

1. Accede a la **web oficial de Odoo** y localiza el **instalador para Windows**.
		Accedemos al enlace que le otorgare mas adelante para descargar Odoo Comunnity para su Windows 10  
		https://www.odoo.com/es_ES/page/download  
		Una vez en la pagina, pondremos nuestros datos como empresa y bajaremos la ultima versión de Community para Windows  
   - ![Pagina Descarga](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/03-descarga_instalador/pagina_instalar.png)  
   - Una vez introducida la información bajamos la ultima versión, en este caso la 19.
