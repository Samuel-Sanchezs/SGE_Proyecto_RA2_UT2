# 07 — Ejecución y servicio

Una vez instalado se ejecutara por primera vez de forma automática pero siempre podemos volver a abrirlo de otras formas

1. Inicia Odoo mediante el **servicio de Windows** o acceso directo.
Podremos iniciar el servicio de Odoo buscando su servicio en la aplicacion servicios de Windows e iniciando el servicio **odoo-server-19.0**  
![Servicios](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/07-ejecucion_servicio/servicios.png)  

2. Verifica el estado del servicio (**Servicios** de Windows).
Verificamos el estado del servicio de Odoo  
![Servicio Odoo](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/07-ejecucion_servicio/servicio_odoo.png)  
Al comprobarlo podemos ver que esta en ejecución y también en inicio automático, indicándonos que se iniciara junto al sistema no teniendo que hacerlo manualmente
