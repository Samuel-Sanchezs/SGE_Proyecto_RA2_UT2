# 08 — Creación de base de datos de prueba

1. Accede a `http://localhost:8069` en tu navegador.
Accedemos al localhost para que nos aparezca este primer menu en pantalla
![LocalHost](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/08-creacion_bd_prueba/localhost.png)
2. Crea una **base de datos nueva** (admin email y contraseña).
Introducimos nuestros datos y le ponemos una contraseña al master y a nuestro usuario
3. Elige módulos iniciales si procede.
Una vez hecho todo esto, nos aparecerá el menú de distintas aplicaciones a instalar, yo instalare la aplicación de sitio web como ejemplo, tu puedes instalar las que necesites.  
![Aplicaciones](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/08-creacion_bd_prueba/aplicaciones.png)
