# 05 — Instalación de Odoo

1. Ejecuta el instalador (`.exe`) como **Administrador**.  
		Elegimos el idioma para la instalación  
![Instalador](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/05-instalacion_odoo/idioma.png)  
2. Acepta la licencia y elige **ruta de instalación**.
		De forma predeterminada se nos pondra esta ruta, pero podemos modificarla  
![Instalador](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/05-instalacion_odoo/ruta.png)  
3. Si se solicita, configura la conexión a **PostgreSQL**.
		Nos pedirá un nombre de usuario para el admin y una contraseña  
		Configuraremos el puerto al 8069
![Instalador](/retos/Reto_02_Instalacion_Odoo_Windows_Samuel_Sanchez_Sandoval/assets/img/05-instalacion_odoo/admin.png)  
4. Completa la instalación y finaliza el asistente.
