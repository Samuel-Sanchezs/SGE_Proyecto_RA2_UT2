- **Firma electrónica**: campos (firma/nombre/fecha), envío y registro.  
Accedemos al apartado de firma electronica y le damos usar el ejemplo  
![Firma electronica](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/firma_electronica.png)  
Con el ejemplo creado podemos modificarlo a nuestro gusto, con apartados de nombres, fechas, firmas, etc..  
![Ejemplo firma](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/ejemplo_firma.png)  
Una vez completado podremos darle a firmar ahora para rellenarlo y enviarlo  
![Firmar](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/firmar.png)

![Enviar](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/enviar.png)  
- **Información (Knowledge)**: wiki tipo Notion, permisos y publicación.   

Abrimos la aplicación en la cual podemos ver una nota de bienvenida y ejemplo  
![Informacion](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/informacion_app.png)  
Vamos a darle al mas a la izquierda y crearnos un apartado de trabajo  
![Apartado](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/crear_apartado_trabajo.png)  
Una vez creado vamos a modificarlo para que tenga portada, un titulo y porque no un emoji  
![Portada](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/portada.png)  
Para ello tendremos que darle a los tres puntos arriba a la derecha y activar el fondo y el emoji, se nos pondrán dos de forma predeterminada  
![Apartado Portada](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/apartado_portada.png)  
Aparte de todo esto podemos usar distintos comandos para crear títulos, índices, enlaces y demás en nuestro apartado de trabajo, con tan solo poner una "/" se desplegaran nuestras opciones  
![Comandos](/retos/Reto_01_Manual_Odoo_Samuel_Sanchez_Sandoval/assets/img/09-documentos_firma_info/comandos.png)
